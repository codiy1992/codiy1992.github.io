阿里云ECS扩容系统盘

codiy 2019-03-05 15:37:33
01-DevOPs

详见 扩展分区和文件系统_Linux系统盘

针对centos7系统，操作如下，注意是系统盘, 不是数据盘:

# 安装扩容工具
yum install cloud-utils-growpart
yum install xfsprogs
# 确定内核版本大于3.6.0
uname -a
